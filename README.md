# Предметная область
Поездки. Каждая поездка может состоять нескольких остновок. <br>
Отсановка совершается в городе в опредлённое время. <br>
Город находится в регионе некой страны, у него своя часовая зона. <br>

Приложение реализует показа следущей ифнормации:
* Города - отсортированные по часовому поясу
* Города - поиск по названию
* Маршруты - отсортированные по количеству остановок
* Маршруты - проходящие через конкретный город
* Количество остановок по городам
